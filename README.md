# Buttons Experiment

A cool game, a bit similar to Death Penalty. Has minigames with buttons, you gotta read the text and then do the challenge that appears on the button. Like repeat the sequence, simon says, voting, duel, etc. It's a social game so it needs at least 10 players to get at least some good experience, ideally 20 players though.

Game is far to be fully finished as it lacks of FOMO, customization and ofc players. But from early feedback I can say that people really love to play it when there are many people joining at the same time, or when streamers are joining to try. But omg it's really hard to make this game, especially balancing is hard, people are saying that it lacks of variety though it has a lot of things, so I think minigames are too similar to each other, so I will be improving this part of the game by adding some more visual differences between minigames.

If you're willing to try the game it's on <https://www.roblox.com/games/115379867051786/Buttons-Experiment>. But you can also analyze the code and think if it's good or not.
Make sure you join at least with somebody as the game is empty lately, the minimal amount of people is 3 players, but with 3 players it's not that cool.
